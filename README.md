# cybersecurity-lab-jon-bueno
Home Lab de Ciberseguridad (Blue Team / Red Team) orientado a práctica, documentación y desarrollo de entornos virtualizados.

## Objetivo
Crear un entorno controlado donde practicar ataques y defensas usando Kali Linux (atacante)
y Ubuntu (defensa), con documentación técnica del proceso.

## Estructura del repositorio

- `docs`: documentación técnica y reportes
- `.gitattributes`: configuraciones sobre como git debe manejar ciertos archivos.
- `.gitignore`: indicamos a Git que archivos o carpetas no deben incluirse. 
- `LICENSE`: definimos los términos legales mediante los cuales personas externas al proyecto pueden modificar o distribuir el código.
- `README`: documento de presentación de proyecto.

## Como reproducir
1. Clonar el repositorio.
2. Crear máquinas virtuales (Kali y Ubuntu).
3. Seguir la guía en `docs/` para realizar las pruebas.
4. Añadir los resultados (pcaps, logs, etc.) al repositorio.
